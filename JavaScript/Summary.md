## 구조 분해 할당 (Destructing assignment)
구조 분해 할당이란 배열 또는 객체의 속성을 해체하여 그 값을 개별 변수에 담을 수 있게 하는 Javascript 표현식 입니다. 이를 통해 기본값 할당, 변수 교환, 객체나 배열의 요소 추출, 함수 매개변수 작업 등 복잡한 데이터 구조를 더 간결하게 처리할 수 있습니다.
